## 中文版本

### 隐私政策

密枢（Key Core）是一款专为 macOS 平台设计的 AI 密钥管理应用。我们非常重视您的隐私和数据安全。

#### 数据收集

**我们不收集任何用户数据。**

密枢应用完全在您的设备上本地运行，所有数据都存储在您的 Mac 上，不会上传到任何服务器。

#### 数据存储

- **本地存储**：所有密钥数据、配置信息和用户设置都存储在您的 Mac 本地
- **加密存储**：使用 AES-256-GCM 加密算法保护您的密钥数据
- **Keychain 集成**：加密密钥存储在 macOS Keychain 中，确保安全性

#### 数据访问权限

应用需要以下权限以提供核心功能：

- **文档文件夹访问**：用于选择配置目录
- **下载文件夹访问**：用于选择配置目录
- **网络访问**（可选）：用于检查配置更新，您可以随时禁用此功能

#### 数据使用

- 所有数据仅用于应用功能，不会用于任何其他目的
- 我们不会分析、分享或出售您的任何数据
- 我们不会将您的数据用于广告或营销

#### 数据导出和删除

- 您可以随时导出您的数据（加密格式）
- 您可以随时删除应用和所有数据
- 删除应用后，所有数据将从您的设备中移除

#### 第三方服务

- 应用不包含任何第三方分析或广告服务
- 应用不会与任何第三方服务器通信（除了可选的配置更新检查）

#### 联系我们

如果您对隐私政策有任何疑问，请通过以下方式联系我们：

- **邮箱**：liuhuayao@126.com
- **GitHub**：https://github.com/liuhauyao/key-core

## English Version

### Privacy Policy

Key Core (密枢) is an AI key management application designed for macOS. We take your privacy and data security very seriously.

#### Data Collection

**We do not collect any user data.**

Key Core runs entirely on your device, and all data is stored locally on your Mac. No data is uploaded to any server.

#### Data Storage

- **Local Storage**: All key data, configuration information, and user settings are stored locally on your Mac
- **Encrypted Storage**: Uses AES-256-GCM encryption algorithm to protect your key data
- **Keychain Integration**: Encryption keys are stored in macOS Keychain for enhanced security

#### Data Access Permissions

The application requires the following permissions to provide core functionality:

- **Documents Folder Access**: Used to select configuration directory
- **User-selected File Access**: Used for importing/exporting configuration files when you explicitly choose them
- **Network Access** (Optional): Used to check for configuration updates. You can disable this feature at any time.

#### Data Usage

- All data is used solely for application functionality and for no other purpose
- We do not analyze, share, or sell any of your data
- We do not use your data for advertising or marketing

#### Data Export and Deletion

- You can export your data at any time (in encrypted format)
- You can delete the application and all data at any time
- After deleting the application, all data will be removed from your device

#### Third-Party Services

- The application does not include any third-party analytics or advertising services
- The application does not communicate with any third-party servers (except for optional configuration update checks)

#### Contact Us

If you have any questions about this Privacy Policy, please contact us:

- **Email**: liuhuayao@126.com
