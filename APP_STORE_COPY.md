密枢是一款专为 macOS 设计的 AI 密钥管理工具，帮助您安全、便捷地管理各类 AI 服务的 API 密钥。采用 AES-256-GCM 加密，数据完全本地存储，支持过期提醒、快速搜索、一键复制等功能。特别支持一键切换 Claude Code 和 Codex 的供应商配置，让您在官方和第三方供应商之间快速切换，提升工作效率。

Features include expiration reminders, quick search, one-click copy, and support for multiple AI platforms. Especially supports one-click switching between providers for Claude Code and Codex, allowing you to quickly switch between official and third-party providers.

## 六、隐私政策和技术支持网址

### 隐私政策要点

应用需要访问以下权限：
- 文档文件夹：用于选择配置目录
- 下载文件夹：用于选择配置目录
- 网络访问：用于检查配置更新（可选）

所有数据均存储在本地，不会上传到任何服务器。

### 技术支持网址（必需）

**重要提示：** App Store Connect 要求必须提供技术支持网址（Support URL）。这是一个必需字段。

**要求：**
- 必须是一个有效的、可访问的网页地址
- 建议包含以下内容：
  - 应用功能介绍
  - 常见问题解答（FAQ）
  - 联系方式（邮箱或联系表单）
  - 使用说明或帮助文档

**建议方案：**
1. **GitHub Pages**：在 GitHub 仓库创建 `docs` 或 `support` 页面
2. **个人网站**：如果有个人网站，创建支持页面
3. **博客平台**：在 CSDN、简书等平台创建技术支持文章
4. **GitHub README**：使用 GitHub 仓库的 README 作为支持页面

**示例格式：**
- `https://github.com/yourusername/key-core/wiki/Support`
- `https://yourdomain.com/support`
- `https://yourusername.github.io/key-core/support`

**注意：** 如果技术支持网址无效或无法访问，应用审核可能会被拒绝。
